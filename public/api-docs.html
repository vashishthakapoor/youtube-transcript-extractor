POST

/api/transcript

Fetch transcript for a YouTube video

Request Body

{
  "videoId": "J1jm4MoZw5Y",
  "language": "en",
  "origin": "auto_generated"
}

Response

{
  "success": true,
  "videoId": "J1jm4MoZw5Y",
  "language": "en",
  "origin": "auto_generated",
  "transcript": "[0s] So then guys...",
  "stats": {
    "segments": 366,
    "words": 2841,
    "characters": 16629,
    "duration": 792
  }
}

Parameters

Parameter Type Required Description
videoId string Yes YouTube video ID (11 characters)
language string No Language code (default: "en")
origin string No uploader_provided | auto_generated

Error Responses

Status Code Description Common Causes
400 Bad Request Invalid video ID format
401 Unauthorized Invalid Oxylabs credentials
404 Not Found Video not found or no transcript
500 Server Error Missing credentials or API error
503 Service Unavailable Network issues or API timeout

Example Usage

JavaScript/Fetch

fetch('/api/transcript', {
  method: 'POST',
  headers: {
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    videoId: 'J1jm4MoZw5Y',
    language: 'en',
    origin: 'auto_generated'
  })
})
.then(response => response.json())
.then(data => console.log(data));

cURL

curl -X POST http://localhost:3000/api/transcript \
  -H "Content-Type: application/json" \
  -d '{
    "videoId": "J1jm4MoZw5Y",
    "language": "en", 
    "origin": "auto_generated"
  }'
